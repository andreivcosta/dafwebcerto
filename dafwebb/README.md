# DAFWEB

Sistema acadêmico desenvolvido para comparação tributária entre Pessoa Física (PF) e Pessoa Jurídica (PJ), permitindo simulações de impostos, cálculos automáticos e análise de renda líquida de forma prática e visual.

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- Node.js
- Express
- jsPDF
- html2canvas

## 📌 Funcionalidades

- Comparação entre tributação PF e PJ
- Cálculo automático de:
  - IRPF
  - INSS
  - Simples Nacional
- Exibição de renda líquida
- Gráficos e visualização de dados
- Exportação de resultados em PDF
- Interface moderna e responsiva

## ⚙️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/joaopedrosgdevv/dafwebb.git
```

Acesse a pasta:

```bash
cd dafwebb
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
